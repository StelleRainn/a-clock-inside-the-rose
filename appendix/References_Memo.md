## 第一章 绪论

- **引用概念/出处**：《深度工作》(Deep Work: Rules for Focused Success in a Distracted World)
- **相关学者/背景**：卡尔·纽波特 (Cal Newport)
- **论文引用位置**：第一章 1.1节 研究背景
- **内容摘要/说明**：在信息技术飞速进步带来的“注意力危机”下，能够进行深度思考与专注工作的能力（即“深度工作”）将成为个体最稀缺的核心竞争力。

- **引用概念/出处**：心流理论 (Flow Theory)
- **相关学者/背景**：米哈里·契克森米哈赖 (Mihaly Csikszentmihalyi)
- **论文引用位置**：第一章 1.1节 研究背景
- **内容摘要/说明**：现代数字环境的干扰严重阻碍了人们进入“心流（Flow）”状态，即一种全身心投入、忘我且高效的精神状态。

- **引用概念/出处**：番茄工作法 (Pomodoro Technique)
- **相关学者/背景**：弗朗切斯科·西里洛 (Francesco Cirillo)
- **论文引用位置**：第一章 1.1节 研究背景 / 1.2节 国内外研究现状
- **内容摘要/说明**：作为一种经典的时间管理方法被广泛推崇，但在现有工具中往往与任务规划脱节。

- **引用概念/出处**：GTD (Getting Things Done) 理念
- **相关学者/背景**：戴维·艾伦 (David Allen)
- **论文引用位置**：第一章 1.2.1节 国外研究现状
- **内容摘要/说明**：Todoist、Things 等软件强调 GTD 理念的落地，帮助用户高效收集与整理任务。

- **引用概念/出处**：自我量化 (Quantified Self)
- **相关学者/背景**：连线杂志编辑 Gary Wolf 和 Kevin Kelly 于 2007 年提出
- **论文引用位置**：第一章 1.2.1节 国外研究现状 / 1.3.1节 研究目标
- **内容摘要/说明**：指利用技术手段获取个人日常生活中的数据，现有专注应用数据分析维度有限，难以满足专业用户对自我量化的深度需求。

## 第五章 系统实现

- **引用概念/出处**：层叠样式表变量 (CSS Custom Properties / CSS Variables)
- **相关学者/背景**：W3C (World Wide Web Consortium) 制定的 CSS 规范
- **论文引用位置**：第五章 5.4.1 全局深色模式的无缝切换
- **内容摘要/说明**：利用 CSS 变量的层叠与继承特性，实现了无需 JavaScript 干预的极低开销、全站丝滑的深色模式主题切换。

- **引用概念/出处**：彩虹表攻击 (Rainbow Table Attack)
- **相关学者/背景**：菲利普·奥克勒（Philippe Oechslin）于 2003 年提出
- **论文引用位置**：第五章 5.5.1 用户认证与鉴权
- **内容摘要/说明**：一种破解哈希加密密码的高效空间/时间权衡技术。本系统通过引入强制加盐的慢速哈希算法（如 BCrypt）来防御此类攻击。
