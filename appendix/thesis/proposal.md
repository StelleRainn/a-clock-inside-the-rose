# 开题报告

## 名称

课题名称：**基于 Spring Boot + Vue 的个人效率中心系统设计与实现**

项目名称：**A Clock Inside the Rose**(ACIR)



## 一、选题背景与意义（为何选这个课题）

随着信息技术的发展与生活节奏的加快，个人在学习、工作和生活中需要同时处理多项事务，如何进行有效的时间管理与任务规划，已成为影响效率与专注力的重要因素。传统的纸质计划或简单的备忘录工具，难以满足用户对任务可视化管理、专注时间统计与长期行为分析的需求。

近年来，番茄工作法（Pomodoro Technique）、任务看板（Kanban）等效率管理理念被广泛应用于个人学习与工作场景中，但市面上的相关效率管理工具多为商业化产品，其功能封闭、定制性不足，不利于系统性学习与二次开发。同时，从教学实践角度看，基于前后端分离架构的 Web 应用开发已成为软件工程类专业的重要能力要求。

因此，本课题以“个人效率管理”为应用场景，结合任务管理、番茄钟计时与数据统计分析等功能，设计并实现一个基于 Spring Boot 与 Vue 的个人效率中心系统。该研究不仅具有一定的现实应用价值，也能够综合训练学生在系统分析、数据库设计、后端业务逻辑实现以及前端交互开发等方面的工程能力，具有较高的实践意义和教学价值。



## 二、国内外研究现状（前人做了什么，还有什么空白）

在国外，个人效率管理与时间管理工具的发展较为成熟，如 Todoist、Trello、Notion 等产品在任务管理与可视化方面已形成较完善的体系。这些系统通常支持任务分类、看板视图以及基础的数据统计功能，但其内部实现细节多为闭源，且针对具体教学或实验场景的系统性研究较少。

在国内，相关研究多集中于任务管理系统、日程管理系统或基于 Web 的学习管理平台，部分文献对番茄工作法在学习效率提升中的作用进行了定性分析，但将任务管理、专注计时与数据统计三者进行系统整合，并以完整 Web 系统形式实现的研究相对较少。

总体来看，现有研究与产品在功能上较为成熟，但在“面向教学实践的完整系统设计”“前后端分离架构下的效率管理系统实现”方面仍存在一定研究与实现空间。本课题在现有研究基础上，侧重系统工程实现与功能整合，具有一定的实践探索意义。



## 三、研究目标与内容（本研究要做什么）

（一）研究目标

本课题的研究目标是设计并实现一个结构清晰、功能完整、可扩展的个人效率中心系统，实现对用户任务、专注时间及效率数据的统一管理与可视化展示，并通过该系统验证基于 Spring Boot + Vue 的前后端分离架构在中小型应用中的可行性。

（二）研究内容

围绕上述目标，系统主要包括以下研究与实现内容：
1.	用户管理功能
实现用户注册、登录、身份认证与基本信息管理功能，保障系统数据的安全性与用户隔离。
2.	任务管理功能
支持任务的创建、编辑、删除与状态管理，任务可设置优先级、标签及截止时间，并按照不同状态进行分类展示。
3.	任务看板功能
通过可视化看板方式展示任务进度，实现“待办—进行中—已完成”等状态流转，提高任务管理的直观性。
4.	番茄钟计时功能
实现基于番茄工作法的专注计时模块，支持自定义专注与休息时长，并记录用户的专注次数与时长数据。
5.	统计分析功能
对任务完成情况与番茄钟数据进行统计分析，生成效率趋势图表，为用户提供行为反馈与总结依据。



## 四、拟解决的关键问题（研究的难点在哪里）
1.	前后端分离架构下的数据交互设计问题
需要合理设计 RESTful 接口，实现前端与后端之间稳定、高效的数据通信。
2.	任务状态与专注数据的关联建模问题
如何在数据库中合理设计任务、番茄钟记录及统计数据之间的关系，是系统实现的关键之一。
3.	数据统计与可视化展示问题
需要对用户行为数据进行整理与统计，并以直观的图表形式呈现，提高系统的实用性与可读性。



## 五、研究方法与技术路线（如何实现研究目标）

（一）研究方法

本课题采用工程实践与系统实现相结合的方法，通过需求分析、系统设计、编码实现与测试验证等步骤完成研究。

（二）技术路线

系统整体采用前后端分离架构，具体技术路线如下：
•	后端部分：
使用 Spring Boot 框架构建后端服务，采用分层架构设计（Controller、Service、DAO），实现业务逻辑处理与数据访问；使用 MySQL 作为关系型数据库进行数据存储。
•	前端部分：
基于 Vue 框架进行界面开发，采用组件化设计方式，实现任务管理、看板展示、番茄钟计时与数据可视化功能。
•	数据交互方式：
前后端通过 RESTful API 进行通信，数据格式采用 JSON。
•	统计与展示：
使用前端图表库对任务与专注数据进行统计分析与可视化展示。



## 六、预期成果（研究完成后能产出什么）
1.	一套基于 Spring Boot + Vue 的个人效率中心系统，实现核心功能并可稳定运行；
2.	系统需求分析文档、系统设计说明文档及数据库设计文档；
3.	完整的毕业论文一篇，系统阐述课题背景、设计思路、实现过程与实验结果；
4.	系统演示与答辩展示材料。
