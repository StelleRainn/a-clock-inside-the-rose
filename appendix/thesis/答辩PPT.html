A Clock Inside the Rose (ACIR) - 蔷 薇 之 刻
答辩人：徐友友

项目背景：针对当下严重的注意力碎片化问题，现有工具常面临任务管控与专注计时相割裂的局面。

解决方案：本课题打破功能壁垒，搭建了基础看板，并引入“沉浸式心流”理念重构专注体验。

数据驱动：借助后端的 SQL 聚合运算，渲染热力图等直观的自我量化反馈。

核心架构设计 (B/S 分层体系)

系统采用前后端分离架构，确保高可用冗余与工程解耦：

  • 表现层 (Frontend)： Vue 3 + Pinia + Element Plus，实现沉浸式 UI 与 ECharts 数据可视化。
  • 网关层 (Gateway)： Nginx 承担静态资源分发与反向代理。
  • 业务层 (Backend)： Spring Boot 3 构建 MVC 架构，JWT + BCrypt 保障安全。
  • 持久与存储层： MyBatis 3.x 配合 MySQL 8.0，实现复杂聚合算力下推。

核心创新点：AI 上下文注入与智能诊断

本课题探索性地利用上下文注入 (Context Injection) 技术对接了生成式大模型 (Google Gemini)。

  • 实时状态感知： 抓取当前未结任务、今日专注时长等私有快照作为上下文喂给 AI。
  • 专属效率诊断： 使得 AI 在一定程度上能感知用户的实时状态，从而提供定制化的调优策略。
  • 工程可行性： 初步验证了沉浸式体验结合智能化干预在生产力工具中的有效落地。

其他工程亮点

  • 基于 Sortable.js 的“挤占式重排”看板算法。
  • 防作弊番茄钟与“禅定模式”UI 防走神机制。
  • Docker 容器化一键部署与数据挂载持久化。
